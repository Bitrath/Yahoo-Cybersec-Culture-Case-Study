# Yahoo! Cybersecurity Culture Case Study📌 

## Project Overview
This repository contains a comprehensive case study analyzing the transformation of Yahoo's organizational cybersecurity culture. The project investigates how Yahoo shifted its strategy from traditional security awareness to data-driven Behavioral Engineering to mitigate human-factor risks.

## 🚨 The Catalyst
The cultural transformation was necessitated by the largest data breaches on record:
- **2013**: A massive breach affecting 3 billion user accounts.
- **2014**: A subsequent breach compromising 500 million user accounts.With an estimated 96% of all security breaches taking place due to human error , Yahoo recognized that "security awareness is no longer enough".

## 🛡️ The Strategy: Behavioral Engineering and The Feedback Control Model
Spearheaded by The Paranoids (Yahoo's internal Security Team) and their Proactive Engagement Team , the company implemented a Feedback Control Model to change employee behavior. The process involved four key steps:
- **Set Strategic Goals**: Identify the desired behavioral goal.
- **Establish Metrics**: Find an appropriate measure, create a baseline, and set performance standards.
- **Compare to Actual Standards**: Use a transparent dashboard to monitor behavior.
- **Take Corrective Action**: Adjust actions over time to affect the measured behavior. To execute this, Yahoo utilized a Hybrid Control Process that combined Hierarchical (monitoring, reward systems) and Decentralized (empowering managers, educational processes) approaches.

## 📊 Key Metrics Monitored
To baseline and influence security behavior, the team focused on specific human-factor metrics:
- **Susceptibility Rate**: The likelihood of employees falling for phishing attempts.
- **Credential Capture Rate**: The rate at which employees submitted passwords to simulated malicious sites.
- **Reporting Rate**: The frequency at which employees actively reported suspicious emails.

## 🛠️ Interventions Implemented
To drive the desired behaviors, Yahoo utilized a mix of Choice Architecture, Communication, and Incentivization.
- **Choice Architecture**: Default installed corporate password managers to all corporate devices to remove guesswork.
- **Communication**: Published a transparent Behavior Dashboard for managers and set up automatic nudge emails for users.
- **Incentivization and Gamification**: Published a recognition program and rewarded top users with Paranoid-branded apparel (T-shirts, hoodies, hats).

## 📈 Results & Impact
The implementation of this data-driven behavioral framework yielded significant improvements in Yahoo's security posture.
- **Credential Capture Dropped**: There was a 50% reduction in employees' credentials captured during phishing simulations.
- **Reporting Increased**: The number of suspicious emails reported by employees doubled.
- **Tool Adoption**: The usage of the company's corporate password manager tripled.
